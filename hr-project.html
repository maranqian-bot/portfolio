Features

주요 기능

직원의 근태와 인사 정보를 한 시스템에서 관리하도록 로그인·인증부터 근태·휴가 관리까지 구성했습니다.

Auth

로그인 · 인증

Spring Security 기반 로그인과 인증 흐름을 구현했습니다. 이메일 인증번호와 Refresh Token을 Redis에 저장해 인증 상태와 세션을 관리하고, 비밀번호 재설정 흐름을 이메일 인증과 연결했습니다.

HR

직원 관리

직원 정보를 등록·조회·수정하고, 부서 배치와 상태를 관리하는 인사 데이터의 기본 흐름을 다룹니다.

HR

부서 관리

부서를 생성하고 직원을 배치하며, 조직 구조를 기준으로 인사 데이터를 정리합니다.

Attendance

근태 관리

출근·퇴근 기록과 근태 현황을 관리하고, 근무일·지각·결근 같은 근태 데이터를 집계합니다.

Attendance

휴가 관리

휴가 신청과 처리 흐름을 관리해 근태 데이터와 연결되도록 구성했습니다.
